1. AWS - это платформа облачных вычислений, которая предлагает различных сервисов от виртуальных серверов, хранилищ и т.д. 
2. Облачные сервисы - это предоставление доступа к IT-ресурсам через интернет по запросу, без необходимости владения и обслуживания собственного оборудования
3. Основные модели облаков: модели предоставления сервисов и модели развертывания.
4. Amazon S3 - это облачный сервис объектного хранения данных от Amazon Web Services, который позволяет хранить и извлекать любые объемы данных в интернете, предлагая высокую производительность, надежность, безопасность и доступность, используя API или веб-интерфейс. 
5. Bucket - это контейнер для хранения данных в облачных хранилищах.
